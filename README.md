# HowToDo

HowToDo is a clean and minimal task management app built with Flutter.
It helps you organize daily tasks, track progress, and stay productive—whether it’s for work, study, or personal projects.

---

## Features

- User authentication (Sign up / Login)
- Create, edit, and delete tasks quickly
- Local notifications for every task
- Offline storage using Hive , and optional online storage
- Dark & Light theme support
- Responsive UI
- Real-time data with Firebase
- Cross-platform: Android & iOS 

---

## Tech Stack

- **Flutter**
- **Firebase** : (Auth, Firestore, Storage)
- **Bloc/Cubit** : state management
- **Hive** : lightweight, fast local database
- **flutter_localizations** : to provide two Languages (en & ar)
- **flutter_screenutil** : full width and size control
